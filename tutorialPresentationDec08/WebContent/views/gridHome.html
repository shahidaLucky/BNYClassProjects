Welcome Css Grid

CSS Grid Layout is the most powerful layout system available in CSS. It is a 2-dimensional system, meaning it can handle both columns and rows, unlike flexbox which is largely a 1-dimensional system. You work with Grid Layout by applying CSS rules both to a parent element (which becomes the Grid Container) and to that element's children (which become Grid Items).

Introduction

Using a grid-view is very helpful when designing web pages. It makes it easier to place elements on the page.

Flexbox consists of various properties that helps us define two types of boxes, flex containers and flex items. The flex containers have one or more flex items and defines how these flex items are position, order, and how they are wrapped. Flex items have the content

A div container can be made a flex container by providing the display property the value flex, when styling the container. Any container inside is consider a flex item display: flex;

Consider this example:

In this example we have one div container with three direct child.

                    .flex_container
                    {
                    
                        background-color: black;
                        
                        width: 40%;
                        
                        height: 350px;
                        
                        margin: 10px auto;
                        
                        color: black;
                    
                    }
                    .flex_item
                    {
                        
                        background-color: beige;
                        
                        width: 100px;
                        
                        height: 100px;
                        
                        text-align: center;
                        
                        font-size: 60px;
                    }

To make the blaxk box a flex container, we just need to apply the display: flex;
